Front-End Web Developer


What is a Front-end Web Devolper?

Websites and web apps are made by front-end developers. Front-end and back-end are different in that the former relates to how a web page appears, whilst the latter refers to how it functions. Consider front-end to be client-side and back-end to be server-side.

What do you need to be a Front-end Web Developer?

The basic languages for front-end development are HTML, CSS, and JavaScript

What are this job's main responsibilites?

The front-end developer's primary duty is user interface. Creating what the user sees on the webpage is all that is required.
